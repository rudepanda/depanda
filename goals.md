# Goals

1. Support the following platforms
    - Windows (NuGet)
    - Mac OSX (Homebrew)
    - Linux
        - Debian-based (Ubuntu, Linux Mint, Elementary)
        - Fedora, RHEL, CentOS
        - Arch Linux
        - Gentoo
    - FreeBSD
    - OpenBSD
2. Keep it simple, and extensible.
3. Be non-intrusive.
